# GS - Desarrollo Web 🚀

Landing institucional desarrollada en **HTML, CSS y JS**, desplegada en **Vercel**.  
Este proyecto demuestra cómo posicionar un negocio en **Google de forma orgánica**, integrando SEO local y presencia en redes sociales.

---

## 🎯 Objetivo
- Mostrar la capacidad de crear **sitios web escalables y optimizados**.  
- Ejemplo práctico de **posicionamiento orgánico** en Google Business Profile.  

---

## ⚙️ Tecnologías
- **HTML5** para la estructura.  
- **CSS3** con animaciones modernas.  
- **JavaScript** para interacción básica.  
- **Vercel** para despliegue rápido y gratuito.
