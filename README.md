# GS - Desarrollo Web 🚀

Landing de ejemplo para el estudio de **HTML, CSS y JS**, desplegada en **Vercel**.  
Página base con aspiraciones de Plantilla, integrando herramientas de SEO para mejorar el posicionamiento en Google.

---

## 🎯 Objetivo
- Mostrar la capacidad de crear **sitios web escalables y optimizados**.  
- Ejemplo práctico de **posicionamiento orgánico** en Google Business Profile.  

---

## ⚙️ Tecnologías
- **HTML5** para la estructura.  
- **CSS3** con animaciones.  
- **JavaScript** para interacción básica.  
- **Vercel** para despliegue rápido y gratuito.
